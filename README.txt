MFQ QUEST v1.0 — PAKET PWA

Isi:
- index.html
- manifest.webmanifest
- sw.js
- icon-192.png
- icon-512.png

Cara mencoba:
1. Ekstrak ZIP.
2. Untuk uji lokal, jalankan melalui web server lokal, bukan sekadar klik file,
   agar service worker/PWA aktif.
3. Unggah semua file ke satu folder hosting HTTPS.
4. Buka alamat index.html di HP.
5. Pilih “Tambahkan ke Layar Utama / Install App”.

Catatan:
- Game tetap dapat dibuka sebagai HTML biasa.
- Progres disimpan di browser/perangkat pengguna.
- Dataset: MFQ v3.2 FINAL, 114 surat.
